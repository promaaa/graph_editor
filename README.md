# Graph Editor

A simple web application for creating and customizing charts in real-time. Built with React.

## Features

-   **Live Editing**: Modify chart data, parameters, and see the results instantly.
-   **Multiple Chart Types**: Switch between Line, Bar, and Pie charts.
-   **Data Table**: Input your data in a simple, table-based format.
-   **Customization**: Change the chart title and axis labels.
-   **PNG Export**: Export your customized chart as a PNG image.

## Getting Started

To run this application locally, follow these steps:

### Prerequisites

You need to have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed on your machine.

### Installation & Running

1.  **Clone the repository** (or download the files).

2.  **Install dependencies**:
    Open your terminal in the project root directory and run:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm start
    ```

This will open the application in your default browser at `http://localhost:3000`.
